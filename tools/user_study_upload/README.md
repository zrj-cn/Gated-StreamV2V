## Results Access and Evaluation Guide

- **Results Overview**: All method results are available and detailed in the `eval.json` file.

- **User Study Setup**: To conduct the user study, run the `python html_dep.py` command. This will create a file named `video_comparison_xxx.html` which you can view in any web browser.

- **Configuring Output**: Adjust the `compare_method` parameter in the `html_gen.py` script to generate HTML files for different comparison methods.

- **Data Output**: At the conclusion of each user study, a JSON file will be saved with the results.

- **Warning**: Please note that the current code has not been fully optimized and may consume substantial memory, potentially causing system slowdowns or freezes. Use with caution.
